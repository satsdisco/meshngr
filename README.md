# Meshngr

Beautiful mesh messaging for MeshCore LoRa devices.

One app. iOS + Android. Off-grid messaging that doesn't look like it was built in 2005.

## What

A Flutter-based messaging app that connects to MeshCore LoRa radios via Bluetooth Low Energy. Peer-to-peer messaging, public channels, and node discovery — wrapped in a UI inspired by Signal and Telegram.

## Why

MeshCore's protocol and hardware are solid. The existing apps aren't. People building mesh networks for privacy, emergencies, and off-grid communication deserve software that matches the quality of the hardware.

## Stack

- **Flutter** (Dart) — single codebase for iOS + Android
- **BLE** — Bluetooth Low Energy connection to MeshCore companion radios
- **MeshCore Protocol** — lightweight hybrid routing mesh for packet radios

## Status

Early development. Scaffolding phase.
